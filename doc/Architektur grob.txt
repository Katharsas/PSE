Unabhängige Programme:

Crawler Server
-------------------
- benötigt Konfiguration & RSS-Feed links (vorgegeben)
-> erzeugt xml Files


Custom Java Servlet
-------------------
- Übergibt xml-Pfade an Elasticsearch, um Index zu erzeugen
- prüft den Index, entfernt zu ähnliche Einträge aus dem Index


Elasticseach Server
-------------------
Bietet Java-API (für Index Erstellung) und REST-API für Datenauslieferung.


Client Servlet
-------------------
Liefert Website, welche Rest-API von Elasticsearch queryd.
