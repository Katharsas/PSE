Unabhängige Programme:

-------------------
Crawler Anwendung (RSSCrawler)
===
- benötigt Konfiguration & RSS-Feed links (vorgegeben)
-> erzeugt xml Files

-------------------
ElasticSearch Server (ESServer)
===
- ElasticSearch Cluster Server (mit Lucene DB) bietet REST-API zur Kommunikation an.

-------------------
Custom Java Anwendung (ESFeeder)
===
- Übergibt xml-Inhalte an ESServer über Java-API, um Index & DB zu erzeugen
- prüft den Index, entfernt zu ähnliche Einträge aus dem Index

-------------------
Client Servlet (CustomerWebApp)
===
- Holt/Sendet Suchanfragen/-ergebnisse vom ESServer über Java-API
- Liefert Website sowie Suchergebnisse an Website
